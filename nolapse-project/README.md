## What Is This?

This workspace contains all product strategy, competitive analysis, and business documentation for Nolapse — a SaaS tool that treats test coverage as governed enterprise infrastructure.

The goal is to build Nolapse as a **neutral, ecosystem-first governance layer** positioned for acquisition by one of four strategic buyers: Sentry/Codecov, SonarSource, GitHub (Microsoft), or GitLab.

## The One-Line Strategy

> Build Nolapse as the neutral coverage governance layer that makes every major platform better — then let the acquisition urgency come to us.

## Key Strategic Decisions (Summary)

- **Exit:** Acquisition by Sentry/Codecov, SonarSource, GitHub, or GitLab
- **Positioning:** Neutral governance layer — complements, never competes
- **Buyer:** Platform / DevOps Teams (primary), VP Eng (secondary), Security/Audit (tertiary)
- **Competitive play:** "Switzerland Model with a Timer" — partner broadly, build traction, create urgency
